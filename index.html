Getting Started

API Host

Todas las peticiones que se realicen a la API de Mensagia deben ser enviadas a la siguiente URL:

https://carcentral.mx/inventory/api/auth/

Interactuar con la API

Si quieres probar nuestra API de una manera rápida te recomendamos el uso de PostMan. Postman es una extensión de Google Chrome que permite interactuar con HTTP API's de forma sencilla a través de una interfaz amigable para construir peticiones y obtener respuestas.

Codigos de estado de respuesta HTTP

La API de Mensagia utiliza los Códigos de estado de respuesta HTTP más utilizados.

Puedes considerar todas las respuestas que no sean devueltas con un código HTTP 200 como un error.

Formato de respuestas de la API

Siempre que realices una petición correcta a la API obtendrás un código de respuesta HTTP 200

Ejemplo de respuestas

                                    {
                                        "data": {
                                        }
                                    }

Si la petición es correcta, te devolveremos la siguiente estructura

                                    {
                                        "errors": {
                                        }
                                    }

Si la petición es incorrecta, te devolveremos la siguiente estructura

Agregar Seminuevo

Este método te permite crear un anucio de un seminuevo

HTTP Request

PUT https://carcentral.mx/inventory/api/auth/addPreowned

Query Parameters
Parameter Description
nameBrand string (required) Example: chevrolet
Nombre de la marca
descriptionBrand string Example: Esta es la descripción de la marca chevrolet
Breve descripción opcional
veh_tipoauto string (required) Example: Aveo 6 PTAS
Descripción de la unidad
veh_anmodelo int (required) Example: 2020
Año modelo
veh_numserie string (required | min:17 | max:17) Example: AN48JIEALO9123SJE
VIN o número de serie del vehículo
veh_kilometr int (required) Example: 20000
Kilometraje
veh_colorexte string (required) Example: rojo
Color exterior
veh_colorinte string (required) Example: negro
Color interior
veh_stransmision string (required | min:3 | max:3) Example: aut
Si el vehículo es manual se llena el campo con "MAN" y si el vehículo es automatico se llema con "AUT"
veh_sfecadqui string (required | date_format:Y-m-d) Example: 2020-11-30
Fecha de adquisición, este campo se llena con formato año, mes, día por ejemplo: 2020-01-01
origen string (required) Example: ABP
CODIGO DE LA EMPRESA
veh_simppvta int (required) Example: 200000
Precio del vehículo en la página
carline string (required) Example: aveo
EL CARLINE, corresponde al grupo de cada vehículo

                                "created": true,
                                "id": 207
                            }
